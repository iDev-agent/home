三步开始用

1

安装

用上面的一键命令，或下载 .dmg 拖入安装。首次打开若提示「已损坏 / 不受信任」，右键图标 → 打开即可。

2

获取 token

飞书私聊 iDev 机器人，发送 /inbox，会回一张带专属 token 的卡片（永久有效、请勿泄露）。

3

填入设置

点菜单栏 iDev 图标 → 右下角齿轮 → 把 token 填进去保存。服务地址自动读取，无需手填。

支持 macOS（Apple Silicon / Intel）与 Windows。已安装的版本会通过内置更新器自动升级到最新。
