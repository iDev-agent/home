你在飞书 @iDev 发起的任务，完成即在 macOS 菜单栏提醒，点一下直达机器人回复的那条消息。

curl -fsSL https://raw.githubusercontent.com/iDev-agent/idev-inbox-dist/main/install.sh | bash

它能做什么

极轻的菜单栏小应用，专注一件事：让你不漏掉任何一个跑完的任务。

🔔

完成即提醒

菜单栏常驻，任务跑完图标旁出现数字角标，不用一直盯着飞书。

🎯

一点直达

点任意一条已完成任务，直接唤起飞书并定位到机器人回复的那条消息。

🗂

自动归档

完成的任务按你设定的时长（10 分钟 ~ 1 天）自动清理，列表始终清爽。

🧰

内置小工具

「绕过签名」等实用工具内置在设置页，解决未签名 App 打不开等问题，持续扩展。

三步开始用

装好后大约一分钟完成配置。

安装

用上面的一键命令，或下载 .dmg 拖入安装。首次打开若提示「已损坏 / 不受信任」，右键图标 → 打开即可。

获取 token

飞书私聊 iDev 机器人，发送 /inbox，机器人会回一张带专属 token 的卡片（永久有效、请勿泄露）。

填入设置

点菜单栏 iDev 图标 → 右下角齿轮 → 把 token 填进去保存。服务地址自动读取，无需手动填。

目前仅支持 macOS（Apple Silicon）。已安装的版本会通过内置更新器自动升级到最新。
