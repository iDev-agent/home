# iDev 收件箱 · 分发

iDev 收件箱（macOS 菜单栏应用）的**官网与安装包**分发仓库；源码在独立的私有仓库，不在此处。

- 官网：https://idev-agent.github.io/idev-inbox-dist/
- 一键安装：`curl -fsSL https://raw.githubusercontent.com/iDev-agent/idev-inbox-dist/main/install.sh | bash`
- 安装包：见 [Releases](https://github.com/iDev-agent/idev-inbox-dist/releases)

安装包由源码仓库 CI 构建后自动发布到此处。
